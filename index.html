# Privasi-bot Kebijakan Privasi Bot Cari Gambar

KEBIJAKAN PRIVASI BOT CARI GAMBAR

Bot ini dibuat untuk mencari dan menampilkan gambar publik dari Pinterest.

Pengumpulan Data: Tidak ada data pribadi pengguna yang dikumpulkan atau disimpan.

Penggunaan Data: Hanya mengambil konten publik dari Pinterest untuk hasil pencarian.

Hubungi: Melalui akun GitHub terkait.
